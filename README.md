# termux-misscallious

How to install?


A: Run


```
apt update -y

apt upgrade -y

apt install curl wget tar coreutils -y

curl -sL https://raw.githubusercontent.com/tanazd1-octo/termux-miscallious/main/install.sh | bash

```

```c```: a c shell similar to cling
```cdt```: run commands inside a directory and go back to your ```$OLDPWD```
```ns```: run scripts with no shebang with the format ```ns <INTERPRETER> <SCRIPT>```
```su```: A fake su wrapper
```tc```: parrarel job runner
```tct```: test if tc works
```x```: universal extractor
